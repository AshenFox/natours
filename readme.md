# Natours

Интерактивная лендинг страница туристического интернет магазина.

## О проекте

Современная лендинг страница с множеством анимированных элементов, меню, видео элементами, модальными окнами, формой заказа.

## Технологии

- HTML
- SCSS
- JavaScript
